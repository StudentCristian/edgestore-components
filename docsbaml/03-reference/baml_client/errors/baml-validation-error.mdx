The `BamlValidationError` class represents an error that occurs when BAML fails to parse or validate LLM output.

## Type Definition

<CodeBlocks>
```typescript Type Definition
class BamlValidationError extends Error {
  type: 'BamlValidationError'
  message: string
  prompt: string
  raw_output: string
}
```
</CodeBlocks>
